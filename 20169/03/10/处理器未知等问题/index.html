##Q1. 完成安装后启动系统黑屏解决办法

####A1.CLOVER缺少支持RX470 kext导致，将下列文件copy到CLOVER中。

kext下载地址

Lilu.kext

WhateverGreen_v1.2.7.kext

##Q2.安装完成后不可以关机，点击关机后出现重启现象。热插拔usb也会造成电脑重启现象。

####A2.原因是缺少USB驱动导致.解决办法为挂载EFI分区，在终端中执行以下命令

curl -s -o ~/Desktop/ryzenusbfix.sh https://raw.githubusercontent.com/XLNCs/ryzenusbfix/master/ryzenusbfix.sh && chmod +x ~/Desktop/ryzenusbfix.sh && ~/Desktop/ryzenusbfix.sh
或者

git clone https://github.com/XLNCs/ryzenusbfix.git ~/Desktop/ryzenusbfix &&chmod +x ~/Desktop/ryzenusbfix/ryzenusbfix.sh &&~/Desktop/ryzenusbfix/ryzenusbfix.sh
执行脚本成功后会有提示，重启电脑即可解决。

github脚本参考

有点编程常识的人都知道，关于本机显示的只是字符串而已，需要更改的话很简单。google教程一抓一大把

在这里说一个简单有效的方法、请阅读这个教程 如何自定义Mac的“关于本机”部分
